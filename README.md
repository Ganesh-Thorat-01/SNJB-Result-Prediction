## Run Locally

Clone the project

```bash
  git clone https://github.com/Ganesh-Thorat-01/SNJB-Result-Prediction
```

Go to the project directory

```bash
  cd SNJB-Result-Prediction

```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python main.py runserver
```
